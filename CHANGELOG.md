# Changelog

## [1.0.0] - 2024

### Added
- Initial release of Lab Cripto
- 6 cipher algorithms:
  - Caesar cipher
  - Monoalphabetic substitution
  - Playfair cipher
  - Vigenère cipher
  - Vernam cipher (One-Time Pad)
  - Columnar transposition
- Professional UI with dark theme (Catppuccin Mocha)
- Step-by-step calculation visualization
- Modular cipher architecture
- Comprehensive documentation
- MIT License
- Example usage script

### Features
- Real-time encryption with intermediate values
- Educational format with detailed calculations
- Input validation and error handling
- Clean and intuitive dark-themed interface
- Responsive layout

### Technical
- Python 3.9+
- Tkinter GUI
- No external dependencies beyond Python stdlib
- PyInstaller support for executable builds
